# 《附录1：Eclipse常用快捷键》

### 1. 查看类层次结构

    Ctrl + T

### 2. 查看本地历史版本

    右击 > Local History

### 3. 组织导入包

    菜单 > Source > Organize Imports    快捷键(Ctrl + Shift + O)

### 4. 删除一行

    Ctrl + D

### 5. 注释某一行代码

    Ctrl + /

### 6. 在Eclipse查看API源代码时，通过Ctrl + 左键进入某个方法的Declaration后，想回退

    Alt + Left/Right
